# Project Scope

## Project Name

Microsoft 365 Security Governance Lab

## Project Objective

To assess and improve the cybersecurity posture of Ari & Sons Pty Ltd using Microsoft security technologies and governance best practices.

## Scope

### In Scope

- Identity and Access Management
- Microsoft Entra ID
- Multi-Factor Authentication
- Conditional Access
- Administrative Access Controls
- Email Security
- Incident Response Planning

### Out of Scope

- Physical Security
- Third-Party Vendor Audits
- Network Infrastructure Security

## Expected Outcomes

- Improved account security
- Reduced phishing risk
- Better access controls
- Improved incident response capability
- Security governance documentation
